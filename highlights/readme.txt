Add highlights section in posts with facebook and twitter sharing 

== Description ==

ReadyMadeWeb Highlights Plugin add highlights section in posts with facebook and twitter sharing.

Features:

*   A Highlights_Widget for adding highlights to sidebar.
*   Highlights positons: Top/Bottom of the post content or hide from post.
*   Highlight Metabox uses wordpress rich text editor

== Installation ==

The quickest method for installing the importer is:

1. Upload the highlights folder to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Settings screen, click on 'ReadyMadeWeb Highlights Settings'
